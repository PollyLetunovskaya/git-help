# Преоект git-help
## Задачи проекта

Помочь начинающим разработчикам с Git

---

**Список основных команд**
1. pwd - в какой папке находимся
2. cd - поменять папку
3. ls - вывести содержимо директории
4. touch - создать файл
5. mkdir - создать директорию
6. cp - копирование файлов
7. cat - чтение файлов

---

**Создать репозиторий**
```
mkdir my_project
cd my_project
git init
```
